google.cloud.bigquery.dataset.DatasetReference.from_string#

classmethod DatasetReference.from_string(full_dataset_id)[source]#

Construct a dataset reference from fully-qualified dataset ID.

Parameters:full_dataset_id (str) – A fully-qualified dataset ID in standard SQL format. Must included both the project ID and the dataset ID, separated by ..
Returns:Dataset reference parsed from full_dataset_id.
Return type:DatasetReference

Examples

>>> DatasetReference.from_string('my-project-id.some_dataset')
DatasetReference('my-project-id', 'some_dataset')
Raises:ValueError – If full_dataset_id is not a fully-qualified dataset ID in standard SQL format.
